# TypeFlow

A typing speed test that runs entirely in the browser, using a word list you provide.

## Features

- Bilingual interface (English / French), switchable at any time
- Load any plain text word list (one word per line)
- Difficulty levels based on word length, plus a custom range option
- Time-based or word-count-based test modes
- Accent-insensitive typing: accented characters in your word list are normalized, so you never need to type them
- Live WPM and accuracy tracking
- Local history and best scores, stored in your browser (no server, no account)
- Export results as a text file

## Usage

1. Open `typing.html` in a browser.
2. Choose your interface language.
3. Load a `.txt` word list (one word per line).
4. Pick a difficulty level and a test mode.
5. Type the displayed words as they appear.

All data stays on your machine. Nothing is uploaded or sent anywhere.

## Requirements

A modern web browser. No installation, build step, or server required.

# TypeFlow (Français)

Un test de vitesse de frappe qui fonctionne entièrement dans le navigateur, à partir d'une liste de mots que vous fournissez.

## Fonctionnalités

- Interface bilingue (français / anglais), modifiable à tout moment
- Chargement de n'importe quelle liste de mots en texte brut (un mot par ligne)
- Niveaux de difficulté basés sur la longueur des mots, avec une plage personnalisable
- Modes de test basés sur le temps ou sur le nombre de mots
- Frappe insensible aux accents : les caractères accentués de votre liste sont normalisés, vous n'avez donc jamais besoin de les taper
- Suivi en direct de la vitesse (MPM) et de la précision
- Historique et meilleurs scores conservés localement dans le navigateur (sans serveur, sans compte)
- Export des résultats au format texte

## Utilisation

1. Ouvrez `typing.html` dans un navigateur.
2. Choisissez la langue de l'interface.
3. Chargez une liste de mots au format `.txt` (un mot par ligne).
4. Sélectionnez un niveau de difficulté et un mode de test.
5. Tapez les mots affichés au fur et à mesure.

Toutes les données restent sur votre machine. Rien n'est envoyé ni transmis ailleurs.

## Prérequis

Un navigateur web récent. Aucune installation, compilation ou serveur n'est nécessaire.
